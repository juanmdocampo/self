SELF is a hybrid platform for accessing therapy/mental health professionals.

**Two UX paths (the "híbrido" concept):**
1. Explore freely — browse psychologist profiles, filter by specialty/price/modality (Tinder-like, already built)
2. Personalized recommendation — user fills a short form, SELF recommends the best match within 24 hours (NOT yet built)

Both paths converge into a simple, fast, user-centered experience.

**Key differentiator:** Unlike traditional directories that just list options, SELF actively guides the user — reducing friction and improving conversion. Verified professionals only.

**Business model:** Commission per session, referral fee, future psychologist subscription (marketplace model).

**Current state:** MVP phase — validating model via manual matching + first professional profiles.

**Roadmap:** Initial validation → directory → automated matching → tech scaling.

**Founder background:** Psychology + HR experience — strong understanding of human behavior.

**Why this matters for code:** The "recommendation path" (form → 24h match) is a major feature not yet in the codebase. This is the next big thing to build after the explore/swipe MVP.
