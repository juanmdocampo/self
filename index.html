📦 Modelos principales models.py

# apps/users/models.py
from django.contrib.auth.models import AbstractBaseUser, BaseUserManager
from django.db import models

class UserType(models.TextChoices):
    PATIENT    = 'patient',    'Paciente'
    PSYCHOLOGIST = 'psychologist', 'Psicólogo/a'
    PROVIDER   = 'provider',   'Proveedor (futuro)'

class User(AbstractBaseUser):
    email      = models.EmailField(unique=True)
    first_name = models.CharField(max_length=100)
    last_name  = models.CharField(max_length=100)
    user_type  = models.CharField(max_length=20, choices=UserType.choices)
    avatar     = models.ImageField(upload_to='avatars/', blank=True)
    created_at = models.DateTimeField(auto_now_add=True)
    is_active  = models.BooleanField(default=True)

    USERNAME_FIELD = 'email'

class PsychologistProfile(models.Model):
    user         = models.OneToOneField(User, on_delete=models.CASCADE)
    license_num  = models.CharField(max_length=50)       # N° matrícula
    specialty    = models.CharField(max_length=100)
    approach     = models.CharField(max_length=100)      # TCC, Gestalt, etc.
    bio          = models.TextField()
    price_min    = models.DecimalField(max_digits=8, decimal_places=2)
    modality     = models.CharField(max_length=20)         # online/presencial/ambas
    location     = models.CharField(max_length=200, blank=True)
    is_verified  = models.BooleanField(default=False)
    rating_avg   = models.FloatField(default=0)

class SwipeAction(models.Model):  # el corazón del match
    patient    = models.ForeignKey(User, related_name='swipes_sent')
    target     = models.ForeignKey(User, related_name='swipes_received')
    liked      = models.BooleanField()
    created_at = models.DateTimeField(auto_now_add=True)

class Match(models.Model):
    patient      = models.ForeignKey(User, related_name='matches_as_patient')
    psychologist = models.ForeignKey(User, related_name='matches_as_psy')
    matched_at   = models.DateTimeField(auto_now_add=True)
    is_active    = models.BooleanField(default=True)

🔌 API Endpoints urls.py

POST/api/auth/register/Crear cuenta (paciente o psicólogo)
POST/api/auth/login/Login → JWT tokens
GET/api/psychologists/Listar psicólogos (con filtros)
GET/api/psychologists/{id}/Detalle de un psicólogo
POST/api/swipe/Registrar swipe (like/nope)
GET/api/matches/Ver matches del usuario
GET/api/discover/Feed de tarjetas (excluye ya vistos)
PUT/api/profile/Actualizar perfil propio
GET/admin/Django Admin (gestión de usuarios)

🚀 Setup rápido

# 1. Crear proyecto
django-admin startproject self_project
cd self_project
python manage.py startapp users
python manage.py startapp matching

# 2. Instalar dependencias
pip install djangorestframework djangorestframework-simplejwt \
            django-cors-headers Pillow psycopg2-binary

# 3. settings.py — agregar apps
INSTALLED_APPS = [
    ...
    'rest_framework',
    'corsheaders',
    'users',
    'matching',
]

AUTH_USER_MODEL = 'users.User'

# 4. Migrar y correr
python manage.py makemigrations
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
